## 🚀 Prérequis
L'application nécessite un Mac sous macOS 11.0 (Big Sur) ou ultérieur. Celle-ci est compatible avec Apple Silicon.
Vous aurez besoin d'1 Mo d'espace disque libre sur votre Mac.
